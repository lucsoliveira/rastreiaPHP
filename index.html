RastreiaPHP

Script para rastreio de encomendas via correios/pac, desenvolvido em PHP, MySQL e HTML

View on GitHub Download .zip Download .tar.gz
  • Script desenvolvido em PHP, MySQL HTML para adição e rastreamento de pacotes Correios/PAC. A intenção desse script é chegar numa versão onde podemos cadastrar usuarios com banco de dados MySQL, e a cada pacote adicionado por qualquer usuario, quando houver atualização na encomenda, o proprio script irá notificar o usuario via e-mail que o pacote teve o destino alterado.
